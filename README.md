***Simple Script To Generate Unlimited Spotify Accounts.***

### [+] Installation

```apt install git python -y```

```cd u-sporti```

```bash install.sh```

```python3 sporti.py```


### Or, Use Single Command
```
apt install git python -y && git clone && cd u-sporti && bash install.sh && python3 sporti.py
```
